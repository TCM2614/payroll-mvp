# Direct Prompt: Financial & Phantom Color Palette

Transform the color palette from emerald green to a financial blue/cyan theme with phantom ethereal effects while maintaining excellent readability.

## Current Palette
- Background: `slate-900` gradient
- Primary: `emerald-500` (too casual for financial app)
- Cards: `bg-black/40` with `border-white/10`
- Text: `text-white/90`, `text-white/70`, `text-white/50`

## Target Palette

### Background
- **Main:** `bg-gradient-to-b from-slate-950 via-slate-900 to-slate-950` (darker, more premium)
- **Alternative:** `from-zinc-950 via-slate-950 to-zinc-950` (phantom feel)

### Cards
- **Primary:** `bg-slate-900/60 border-slate-700/30 shadow-xl shadow-black/50`
- **Secondary:** `bg-slate-800/50 border-slate-600/20`

### Primary Actions
- **Button:** `bg-cyan-500 hover:bg-cyan-600 text-white shadow-lg shadow-cyan-500/30`
- **Alternative:** `bg-teal-500 hover:bg-teal-600 text-white shadow-lg shadow-teal-500/30`

### Text Colors (Readability Optimized)
- **Headings:** `text-slate-50` or `text-white` (high contrast)
- **Body:** `text-slate-200` or `text-slate-300` (better readability)
- **Muted:** `text-slate-400` or `text-slate-500`
- **Labels:** `text-slate-100`

### Accent Colors (Phantom Glow)
- **Primary:** `text-cyan-400 border-cyan-400 ring-cyan-400/30`
- **Secondary:** `text-teal-400 border-teal-400`
- **Premium:** `text-amber-400` (use sparingly)

### Input Fields
- **Background:** `bg-slate-800/60 border-slate-600/40`
- **Text:** `text-slate-100 placeholder:text-slate-500`
- **Focus:** `focus:border-cyan-400 focus:ring-2 focus:ring-cyan-400/30`

### Status Colors
- **Success:** `bg-emerald-500/20 border-emerald-400/30 text-emerald-300` (keep green)
- **Warning:** `bg-amber-500/20 border-amber-400/30 text-amber-300`
- **Error:** `bg-rose-500/20 border-rose-400/30 text-rose-300`
- **Info:** `bg-cyan-500/20 border-cyan-400/30 text-cyan-300`

### Phantom Effects
- **Glow:** `shadow-cyan-500/20 hover:shadow-cyan-500/40`
- **Ethereal borders:** `border-cyan-400/30 hover:border-cyan-400/50`
- **Glass morphism:** `bg-slate-900/60 backdrop-blur-sm border border-slate-700/30`

## Requirements

1. **Financial Theme:** Professional blue/cyan colors (trust, precision, premium)
2. **Phantom Theme:** Ethereal glows, translucent effects, premium feel
3. **Readability:** WCAG AA compliant (minimum 4.5:1 contrast ratio)
4. **Accessibility:** High contrast text, visible focus states
5. **Consistency:** Apply across all components uniformly

## Implementation

Replace all instances of:
- `emerald-500` → `cyan-500` or `teal-500`
- `emerald-400` → `cyan-400` or `teal-400`
- `bg-black/40` → `bg-slate-900/60`
- `border-white/10` → `border-slate-700/30`
- `text-white/90` → `text-slate-50`
- `text-white/70` → `text-slate-200`
- `text-white/50` → `text-slate-400`

## Result

Professional, trustworthy, premium financial application with phantom ethereal aesthetic and excellent readability.
